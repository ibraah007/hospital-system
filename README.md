## 👥 Roles & Permissions

### 🔴 Chief Admin (Default: Ibrahim)
- Full system control  
- Add/remove staff  
- Manage inventory  
- Transfer admin rights  

### 🔵 Official 1
- Generate reports  
- Manage inventory  

### 🟢 Official 2
- Manage staff  

### ⚪ Members
- View dashboard  
- Limited interaction  

---

## 🚀 Features

- ✅ Role-based authentication system  
- ✅ Staff management (add/remove personnel)  
- ✅ Inventory tracking (add/delete items)  
- ✅ Report generation system  
- ✅ Low stock & expired item alerts  
- ✅ Secure PIN update system  
- ✅ Management handover (admin transfer)  
- ✅ Dark mode support 🌙  

---

## 🔄 First-Time Login Process

1. Login using default credentials  
2. Set a new PIN  
3. Use the new PIN for future access
